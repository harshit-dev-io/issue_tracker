This is a backend REST API built to help teams collaborate, track project tasks, and manage software bugs .

The goal of this project is to provide a clean, secure, and organized system for development teams to keep track of what needs to be fixed or built.

Features

1. Collaborate with teams
2. Track issues & projects
3. Real-time sync

🧰 The Tech Stack
Framework: FastAPI 
Database: PostgreSQL
Security: JWT Authentication
